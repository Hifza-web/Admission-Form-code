Jee haan — agar aap text-align: center; ka use karke kisi cheez ko center karna chahti hain, to uska parent element ko center karna zaroori hota hai. .parent { text-align: center; } .child { display: inline-block; /* ya inline bhi chalega */ } ❌ Agar sirf child pe text-align: center; lagaoge: .child { text-align: center; } To kuch bhi center nahi hoga, kyunki text-align sirf parent ke andar content ko control karta hai. Is method sy bi button center ho jata ha: display: block + margin: auto method? display: block; + margin: auto; method kisi bhi block element ko horizontally center kar sakta hai — sirf button tak limited nahi. jis ko comment keya ha is sary ki summary ya ha: ✅ Inline / Inline-Block Elements Ko Center Karne Ke 2 Common Methods: | Method | Kis pe lagta hai? | Kya center karta hai? | | ------------------------- | ------------------------------------ | ---------------------------------------------- | **`text-align: center;`** | Parent element (e.g. `form`, `div`) | Jo cheez **inline** ya **inline-block** ho | | **`margin: auto;+ display: block;`| Child element (e.g. `button`, `img`) | Jo element `**block**` ho **aur width fix** ho | FOR Inline: "Text center karna ho to text-align: center;" "Box center karna ho to margin: auto; + width + block" 📋 Block Element Ko Horizontally Center Karne Ka Summary Table | **Step** | **Detail** | | ------------------ | ---------------------------------------------------------------------------- | | ✅ Element Type | Block-level element hona chahiye (e.g. `div`, `form`, `p`, `h1`, `img`) | | ✅ Display | `display: block;` (ye default hota hai block elements ke liye) | | ✅ Width | `width` fix karni zaroori hai (e.g. `width: 300px;`) | | ✅ Margin | `margin-left: auto; margin-right: auto;` ya shorthand: `margin: auto;` | | 🚫 Width na ho to? | Center **nahi hoga**, kyunki block element by default `width: 100%` hota hai | | ✅ Extra | `text-align: center;` kaam nahi karta block elements pe | 📋 Block Element Ko Vertically Center Karne Ka Summary Table ✅ 1.
